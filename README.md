PriceLogic is a pricing logic reconstruction tool designed specifically for independent artisans in New Zealand. It is more than just a calculator—it is a strategic tool that helps creators find a balance between artistic sensibility and business logic.

Core Logic

Real-time Logical Feedback: All parameter adjustments are reflected instantly on the black "Profit Dashboard."

Customized Survival Stress Test: Users can set monthly targets based on personal living costs (e.g., NZ rent/living expenses).

PDF Report Export: Integrated with html2pdf.js, supporting the export of pricing results as a professional PDF (includes Stripe payment logic integration).

Quick Start

This project is a single-file architecture and requires no environment setup:

Clone this repository.

Open index.html in your browser to start using it.

Deployment Recommendations: Can be deployed for free via GitHub Pages, Vercel, or Cloudflare Pages.

Security & Privacy

100% Client-Side: All calculation logic runs locally in the user's browser; no business data is stored.

Stripe Integration: The payment logic uses only a Publishable Key, ensuring PCI compliance for the payment environment.
